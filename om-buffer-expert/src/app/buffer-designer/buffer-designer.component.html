Quick Start Guide

 Buffer Designer: Generative AI Input

You can now get receipe for your buffer by providing a natural language input. Simply enter the buffer name as you see in the hint like "40mM Phosphate buffer, 0.1M Sodium Chloride, pH 7.0" and the ChatGPT will do the magic, along with the powerful calculation engine to generate a recipe.

To make it even easier, now you can choose a buffer species using autocomplete and it will automatically populate the buffer name for you, which you can easily edit.

Buffer Designer: Classical Input You can also provide input the classic way. The total concentration value you provide will be taken as the first compound's concentration, if the buffering species is not present in both the compounds. pH-Calculator Of course you can also calculate the pH of a solution by providing the compounds and their concentrations. Super Calculator All new! You can now add upto 3 different buffer compounds, one salt compound! In addition, you can have the calculator adjust the pH to a target for you. It will use either Sodium Hydroxide or Hydrochloric Acid to adjust the pH.
